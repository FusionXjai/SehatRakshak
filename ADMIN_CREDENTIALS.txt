═══════════════════════════════════════════════════════════════
                    TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

❌ Cannot Login?
   - Verify email confirmation is disabled in Supabase Dashboard
   - Check Supabase Auth settings
   - Ensure user exists in auth.users table

❌ Wrong Dashboard?
   - Check user role in profiles table
   - Verify Login.tsx redirect logic

❌ No Data Visible?
   - Hospital Admin: Ensure admin_id is set in hospitals table
   - Check Row Level Security (RLS) policies
   - Verify hospital_id foreign key relationships
